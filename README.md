<details>
<summary><b>üîß Technical Deep Dive</b></summary>

<br/>

**Frontend Architecture**
- Framework: Next.js 14 with App Router
- UI: React 19 + TypeScript for type safety
- Styling: Tailwind CSS + CVA for component variants
- State: Zustand for global state, TanStack Query for server state

**3D Graphics Pipeline**
- Engine: Three.js with React Three Fiber wrapper
- Shaders: Custom GLSL fragment shaders for fluid effects
- Optimization: Instanced rendering, GPU particle systems
- Animations: GSAP for timeline-based sequences

**AI Integration**
- Provider: Groq for ultra-low latency inference
- Framework: LangChain for RAG architecture
- Vector Store: Pinecone for semantic search
- Streaming: Server-Sent Events for real-time responses

**Deployment & DevOps**
- Platform: Vercel Edge Network (25+ global regions)
- CDN: Automatic asset optimization & caching
- Analytics: Vercel Analytics + Web Vitals monitoring
